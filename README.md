# Library Books Management System (Core Java)

## Project Overview

This is a **console-based Library Books Management System** built using **Core Java**.
The application allows users to manage books in a library by performing operations such as adding, searching, borrowing, and returning books.

The project demonstrates important **Java backend fundamentals**, including:

* Object-Oriented Programming (OOP)
* Java Collections (HashMap)
* Layered architecture (Model → Service → UI)
* Clean code practices

---

## Features

* Add new books to the library
* View all available books
* Search books by Book ID
* Borrow a book
* Return a borrowed book
* Prevent duplicate Book IDs
* Display book availability status

---

### Description

* **Model Layer**

  * `Books.java` → Represents book data

* **Service Layer**

  * `BookService.java` → Handles all business logic

* **Utility Layer**

  * `Inpututil.java` → Handles user input

* **Main Class**

  * Provides menu and interacts with the user
